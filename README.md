# 🎯 码上猜 CodePop

> 码上猜,马上猜!

Q 弹像素糖果风的**异步好友对战猜图标密码**游戏。纯静态页面,开在 GitHub Pages 上即可和好友远程斗智——不需要任何服务器。

> 经典玩法(猜数字 / 1A2B)升级版:数字变成了 10 个糖果色像素图标。每猜一次,4 个小方块给你打分:**绿块** = 图标对、位置也对;**黄块** = 图标对、位置不对;**灰块** = 没中。用最少的步数破解 4 图标密码!

## 🎮 三种模式

| 模式 | 玩法 |
|------|------|
| ⚔️ **好友对战** | 布置密码 → 生成链接发给好友 → TA 破解后出题反击 → 你迎战 → 自动生成判决书,步数少者获胜。全程异步,微信随时接招 |
| ☀️ **每日挑战** | 全球同题(按日期生成),连胜打卡,表情战报一键分享 |
| 🎲 **无尽练习** | 随机密码,磨炼推理直觉 |

## 🧠 让它更好玩的巧思

- **道具系统(有代价的情报)**:🧲 磁铁问一个图标在不在(+1步)、🔭 望远镜偷看一个位置(+3步)、📺 看 15 秒小广告免费排除一个不在的图标(3 秒后可跳过,跳过没奖励)——用不用道具是每局的策略抉择
- **键盘标记**:长按/右键图标即可标「排除 / 锁定」,像 Wordle 一样减轻记忆负担
- **情报栏**:道具获得的线索自动钉在棋盘上方,异步玩也不怕忘
- **重复猜测拦截**:一模一样的组合会被提醒"再想想",不浪费步数
- **表情战报**:🟢🟡⚪ 无剧透分享格子,朋友圈内卷神器
- **判决书**:对决结束自动生成胜负判决链接,发回去让对手心服口服
- **全程 Q 弹**:果冻按键、翻转揭示、彩带、飞行图标背景、全套原创音效

## 🚀 部署到 GitHub Pages

1. 推送本仓库到 GitHub
2. 仓库 **Settings → Pages → Source** 选 `Deploy from a branch`,分支选 `main`、目录 `/ (root)`,保存
3. 稍等片刻,游戏就在 `https://<用户名>.github.io/CodeGuesser/` 上线了

## 🔧 技术说明

- 纯 HTML/CSS/JS,无构建、无依赖
- 对战数据(密码、战绩)经混淆编码后压进链接的 `#d=` 哈希参数里传递,双方设备各自解码——所以静态托管就能实现异步联机
- 像素图标素材 256px,BoldPixels 像素字体 + 小赖中文字体,本地存档用 `localStorage`

## 🎨 素材

- 图标:Game Icon Pack(256px 白色像素图标)
- 字体:BoldPixels.ttf / Xiaolai-Regular.ttf(小赖字体)
- 音效与广告 GIF:自备素材包
